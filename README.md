## Before publishing

1. In `contact.html`, replace `YOUR-LEAGUE-EMAIL` with the email address that should receive contact-form submissions.
2. In `rules.html`, replace:
   - `YOUR-BYLAWS-PDF-LINK`
   - `YOUR-PLAYING-RULES-LINK`
   - `YOUR-CODE-OF-CONDUCT-LINK`
   - `YOUR-REFUND-POLICY-LINK`
3. Upload every file and the `assets` folder to the root of the GitHub repository.
4. In GitHub, open **Settings → Pages**, choose **Deploy from a branch**, then select `main` and `/ (root)`.
