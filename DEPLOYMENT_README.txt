NWCC Fall Kickball GitHub Pages deployment package

This package includes the full existing website plus:
- forms.html
- coaches-application.html
- background-check.html
- integrated NWCC styling
- mouse/touch/stylus signature pads
- Print / Save as PDF formatting

Deployment:
1. Extract this ZIP.
2. Upload the CONTENTS of the NWCC_Fall-main folder to the root of your GitHub repository.
3. Commit the changes.
4. GitHub Pages will redeploy automatically.

The forms do not submit or store information. Users complete them locally in their browser and print or save as PDF.
